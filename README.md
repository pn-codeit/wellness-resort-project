# wellness-resort-project

University group project: cloud-native web application for a fictional wellness resort with booking configurator, shop, weather integration, and AI features.

## Features
- Wellness vacation configurator with booking functionality
- Wellness shop with shopping cart
- Impressions with images and videos
- Weather display
- AI feature with Gemini
